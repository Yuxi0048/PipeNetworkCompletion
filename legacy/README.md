# Legacy Code

This directory contains notebook-era code kept for auditability and traceability.
The runnable replication path no longer imports from this directory.

- `Inductive.ipynb`: original Colab notebook used for data loading, training,
  evaluation, explanation, and plotting.
- `Module/`: notebook support namespace retained for comparison.
- `linemap.py`: original Folium visualization helper.

Use the root `README.md` and `docs/REPRODUCIBILITY.md` for the maintained
environment and replication commands.
